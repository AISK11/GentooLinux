Flash USB

Download ISO

  1. Download Gentoo ISO from:
    • https://www.gentoo.org/downloads/
  2. Verify download:
    • user$ sha1sum <install-amd64-minimal-YYYYMMDDT170534Z.iso>

USB preparation

  1. Unmount all USB partitions!
  2. Create GPT partition table:
    • root# parted -s </dev/sdX> mktable gpt
  3. Verify new partition table:
    • root# parted </dev/sdX> (p)rint [free]

Flash USB with ISO

  1. Flash ISO to USB:
    • root# dd if=<install-amd64-minimal-YYYYMMDDT170534Z.iso> of=</dev/sdX> [bs=4M | status=progress]

Boot from USB
